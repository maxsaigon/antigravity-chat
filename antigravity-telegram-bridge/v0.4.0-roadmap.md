# Antigravity Telegram Bridge - v0.4.0 Roadmap
## Trọng tâm: Tích hợp Agent Manager & Multi-Project Support

Mục tiêu của phiên bản 0.4.0 là mở rộng khả năng của Telegram Bridge, chuyển từ việc chỉ tương tác với một instance Antigravity duy nhất sang việc quản lý toàn diện **Agent Manager**. Điều này cho phép người dùng khởi tạo, chuyển đổi, và làm việc với nhiều dự án (projects) hoặc ngữ cảnh (workspaces) cùng một lúc thông qua Telegram.

### 1. Kiến trúc đề xuất
- **Agent Registry:** Bridge sẽ duy trì một danh sách các "Agent Sessions" đang hoạt động (ví dụ: Session A cho Project X, Session B cho Project Y).
- **Session Routing:** Mỗi tin nhắn từ Telegram có thể được gán cho một session cụ thể thông qua command hoặc inline keyboard, thay vì luôn gửi vào session mặc định.
- **Tích hợp Native API mới:** Cần mở rộng bộ lệnh API của Antigravity để hỗ trợ `antigravity.createNewAgent(workspacePath)`, `antigravity.switchAgent(agentId)`, và `antigravity.listAgents()`.

### 2. Các tính năng chính (Features)
- **Quản lý Session Đa luồng:**
  - Lệnh `/agents`: Liệt kê tất cả các AI agents/dự án đang chạy. Cung cấp Inline Keyboard để chọn.
  - Lệnh `/new_agent <đường_dẫn_project>`: Khởi tạo một AI session mới cho một thư mục cụ thể.
  - Lệnh `/kill <agent_id>`: Tắt một AI session để giải phóng tài nguyên.
  
- **Chuyển đổi Ngữ cảnh (Context Switching):**
  - Khả năng "ghim" (pin) một project làm mặc định cho các tin nhắn tiếp theo (`/switch <agent_id>`).
  - Hỗ trợ gửi lệnh cho một agent cụ thể mà không cần chuyển mặc định (VD: `/ask <agent_id> <câu_hỏi>`).

- **Đồng bộ Status Đa dự án:**
  - Nâng cấp hệ thống AI Status hiện tại (v0.3.0) để báo cáo trạng thái theo từng agent (VD: `Project A: Typing...`, `Project B: Idle`).

### 3. Thay đổi kỹ thuật (Technical Implementation)
1. **VS Code Extension (src/extension.ts):**
   - Refactor `sendPromptToAgentPanel` để nhận tham số `agentId`.
   - Lắng nghe sự kiện từ Agent Manager thay vì chỉ một panel duy nhất.
2. **Brain Watcher:**
   - Cập nhật cơ chế watch thư mục `~/.gemini/antigravity/brain/`. Có thể mỗi agent sẽ có một thư mục conversation/brain riêng, Brain Watcher cần phân biệt được artifact nào thuộc về project nào.
3. **Telegram UI:**
   - Nâng cấp Inline Keyboards để hiển thị danh sách project/agent rõ ràng hơn.
   - Thêm Context indicator vào tin nhắn phản hồi (Ví dụ: `[Antigravity Chat] 🤖 Response: ...`).
